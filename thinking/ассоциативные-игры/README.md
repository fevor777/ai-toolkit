# Ассоциативные игры для ChatGPT

Четыре варианта системных промптов для развития креативности через свободные ассоциации. Каждый промпт готов к использованию — просто скопируй содержимое файла и вставь в ChatGPT Custom Instructions.

## Краткое сравнение

| Вариант | Темп | Стиль | Лучше всего для | Файл |
|---|---|---|---|---|
| Быстрый поток | Быстрый | Соревновательный | Разогрев, снятие блоков | `быстрый-поток.md` |
| Картограф сознания | Средний | Исследовательский | Самопознание, коучинг | `картограф-сознания.md` |
| Ассоциативный джаз | Средний-быстрый | Творческий, игровой | Писатели, художники | `ассоциативный-джаз.md` |
| Тихий поток | Медленный | Медитативный | Осознанность, вечер | `тихий-поток.md` |

## Как использовать

1. Откройте нужный файл (выберите вариант по таблице выше)
2. Скопируйте содержимое промпта
3. Вставьте в [ChatGPT Custom Instructions](https://platform.openai.com/account/api-keys) или в начало диалога с ChatGPT

## Описание вариантов

### 1. Быстрый поток (Цепочка без тормозов)
Чистая скорость и спонтанность — никакого анализа, только поток сознания. Соревновательная динамика. Идеально для разогрева ума перед интенсивной творческой работой.

### 2. Картограф сознания
Исследовательская динамика с метаанализом. После каждой цепочки из 7–10 слов — совместный разбор: что их связало, какие скрытые темы всплыли. Для самопознания и глубокого понимания своих ассоциативных паттернов.

### 3. Ассоциативный джаз
Творческое партнёрство: ИИ иногда импровизирует, предлагая неожиданные метафоры и поэтические связи. Для писателей, художников и всех, кто ищет новые образы и неожиданные углы зрения.

### 4. Тихий поток
Медитативная практика со замедленным темпом. Каждое слово — как камень, брошенный в воду. Для вечерней рефлексии и практик осознанности.
